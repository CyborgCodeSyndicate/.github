# Code of Conduct

Cyborg Code Syndicate is committed to a respectful, inclusive, and professional environment for everyone interacting with our organization and repositories.

This Code of Conduct applies to participation in:
- All repositories in the Cyborg Code Syndicate GitHub organization (issues, discussions, comments, reviews)
- Pull request discussions (where applicable)
- Any official communication channels referenced by this organization

## Our standards

We expect all participants to:
- **Be respectful**: treat others professionally and with courtesy.
- **Assume good intent**: ask clarifying questions before escalating.
- **Be constructive**: focus feedback on ideas and code, not individuals.
- **Respect privacy**: do not share personal or confidential information.

Unacceptable behavior includes:
- Harassment, discrimination, or hate speech of any kind
- Threats, intimidation, or encouraging others to harass
- Doxxing or sharing private information (including addresses, phone numbers, private emails, employer details)
- Sexualized language or unwanted advances
- Trolling, personal attacks, or persistent disruptive behavior
- Spam, fraud, or malicious content

## Enforcement

Maintainers may take any action they deem appropriate to protect the community, the brand, and the project, including:
- Removing or editing content
- Locking conversations
- Issuing warnings
- Temporarily or permanently restricting participation
- Reporting behavior to GitHub or relevant authorities when required

## Reporting

If you experience or witness unacceptable behavior:
- **Do not engage publicly.**
- Report it privately using the process in **SUPPORT.md**.

If the report involves a maintainer, use the alternate contact route listed in **SUPPORT.md**.
