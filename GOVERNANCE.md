# Governance

Cyber Code Syndicate projects are stewarded by the Cyber Code Syndicate maintainers.

## Roles
- **Steward/Maintainer:** approves changes, manages releases, and sets direction.
- **Contributors:** propose changes via issues and pull requests.

## Decision-making
- Technical decisions are made by maintainers.
- Significant changes should be discussed via an issue before implementation.

## Pre-release note
Until the first public release of each repository, repos may contain confidential pre-release material.
Please follow the confidentiality section in CONTRIBUTING.md.
